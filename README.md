Instantly destroy all glass blocks.

Actually destroying the blocks containing the following tags at a mining speed of `1024.0F`.

`#glasscutter:glasswork`:

* `#c:glass_blocks`
* `#c:glass_panes`

Can be enchanted `minecraft:silk_touch` to get drops.
